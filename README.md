# arduino-RF-433Mhz-ORNO-OR-GB-405
Arduino + RF modul 433Mhz + ORNO OR-GB-405


# explanation code list

A ON  : [11111][0FFFF][F0]

A OFF: [11111][0FFFF][0F]

B ON  : [11111][F0FFF][F0]

B OFF: [11111][F0FFF][0F]

C ON  : [11111][FF0FF][F0]

C OFF : [11111][FF0FF][0F]

D ON  : [11111][FFF0F][F0]

D OFF : [11111][FFF0F][0F]

E ON  : [11111][FFFF0][F0]

E OFF  : [11111][FFFF0][0F]
